Privacy Policy / プライバシーポリシー

Last updated: 2026-05-04

1. 収集する情報 / Information We Collect

JP Bids MCP は、ユーザーの個人情報を収集・保存しません。MCPリクエストに含まれるクエリパラメータ（検索キーワード、都道府県、カテゴリ等）はリクエスト処理にのみ使用し、処理完了後に破棄します。

JP Bids MCP does not collect or store personal information. Query parameters included in MCP requests (search keywords, prefecture, category, etc.) are used only for request processing and discarded after completion.

2. サードパーティへのデータ送信 / Third-Party Data Sharing

  • 中小企業庁 官公需情報ポータルサイト (KKJ): 検索クエリを中小企業庁APIに転送します。中小企業庁のプライバシーポリシーが適用されます。
  • Google Vertex AI: PDF抽出機能使用時、添付資料の内容をVertex AI Geminiに送信します。Google Cloud のデータ処理規約が適用されます。ドキュメントはサーバー側に保存されません。
  • Stripe: Pro tierの従量課金に使用します。Stripe Customer IDのみを送信し、検索内容は送信しません。

Search queries are forwarded to the SME Agency (KKJ) API. When PDF extraction is used, attachment content is sent to Google Vertex AI Gemini for processing. Documents are never stored on our servers. Stripe is used for Pro tier usage billing; only the Stripe Customer ID is transmitted.

3. ログ / Logging

Cloud Run のインフラログ（HTTPステータスコード、レスポンスタイム等）は Google Cloud の標準ログ保持ポリシーに従い保持されます。検索クエリの内容はログに記録しません。

Infrastructure logs (HTTP status codes, response times) are retained per Google Cloud's standard log retention policy. Search query content is not logged.

4. Cookie / Cookies

JP Bids MCP は Cookie を使用しません。

JP Bids MCP does not use cookies.

5. データの保存 / Data Storage

入札情報の添付ファイル（PDF等）は一時的にメモリ上で処理し、レスポンス送信後に即座に破棄します。ディスクやデータベースへの永続化は行いません。

Bid attachment files (PDFs, etc.) are processed in memory only and discarded immediately after the response is sent. No data is persisted to disk or database.

6. お問い合わせ / Contact

プライバシーに関するお問い合わせは GitHub Issues までお願いします。
