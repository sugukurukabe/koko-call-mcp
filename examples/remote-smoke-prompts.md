# Remote Smoke Prompts

## 日本語

MCP Inspector または対応クライアントで `https://mcp.bid-jp.com/mcp` に接続したあと、以下を試してください。

### Tool: search_bids

```json
{
  "query": "システム",
  "prefecture": "鹿児島県",
  "category": "役務",
  "limit": 3
}
```

### Tool: list_recent_bids

```json
{
  "prefecture": "鹿児島県",
  "category": "役務",
  "days": 7,
  "limit": 5
}
```

### Resource Template

```text
prefecture://46
org://鹿児島市
```
