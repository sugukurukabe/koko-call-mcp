# Contributing

## 日本語

PR前に `npm run check` を実行してください。公開済みTool名、必須引数、出力schemaの破壊的変更はADRが必要です。

## English

Run `npm run check` before opening a pull request. Breaking changes to public tool names, required arguments, or output schemas require an ADR.

## Bahasa Indonesia

Jalankan `npm run check` sebelum membuat pull request. Perubahan breaking pada nama tool publik, argumen wajib, atau schema output memerlukan ADR.
