# Reproducible Builds

## 日本語

`npm run build:repro` は `dist/` を削除して2回ビルドし、生成物のSHA-256ハッシュが一致することを確認します。

## English

`npm run build:repro` removes `dist/`, builds twice, and verifies that the SHA-256 hash of the generated artifacts is identical.

## Bahasa Indonesia

`npm run build:repro` menghapus `dist/`, melakukan build dua kali, dan memverifikasi hash SHA-256 artifact yang dihasilkan sama.
