# Mutation Testing

## 日本語

`npm run mutation` は `src/domain` と `src/api` のテスト有効性を検証します。初期しきい値は high 85、low 70、break 60 です。

## English

`npm run mutation` checks test effectiveness for `src/domain` and `src/api`. Initial thresholds are high 85, low 70, and break 60.

## Bahasa Indonesia

`npm run mutation` memeriksa efektivitas test untuk `src/domain` dan `src/api`. Threshold awal adalah high 85, low 70, dan break 60.
