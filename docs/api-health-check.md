# API Health Check

## 日本語

通常の `npm run api:health` はfixtureで正規化処理を確認します。上流APIを実際に確認する場合だけ `RUN_LIVE_KKJ_CHECK=1 npm run api:health` を使います。

## English

The normal `npm run api:health` command verifies normalization with a fixture. Use `RUN_LIVE_KKJ_CHECK=1 npm run api:health` only when checking the live upstream API.

## Bahasa Indonesia

Perintah normal `npm run api:health` memverifikasi normalisasi dengan fixture. Gunakan `RUN_LIVE_KKJ_CHECK=1 npm run api:health` hanya saat memeriksa API upstream secara langsung.
