## 参考: 売却価格の目線

| 形態 | 目線 |
|------|------|
| 完全売却（コード + ドメイン + npm） | 500万〜3,000万円 |
| OEMライセンス（独占） | 月額30〜100万円 |
| OEMライセンス（非独占） | 月額10〜30万円 |
| 技術提携（レベニューシェア） | 売上の10〜20% |

価格根拠: 自社開発コスト（MCP仕様準拠・OAuth・Cloud Run・103テスト）= 推定500〜1,000時間 × エンジニア単価
