# Launch Copy

## 日本語

JP Bids MCP v0.4.0 を公開しました。中小企業庁 官公需情報ポータルサイトの検索APIをMCP化し、Claude/Cursor/VS Codeから日本全国の入札情報を自然言語で検索できます。v0.4.0 では AI Bid Radar 機能群 (`rank_bids`, `explain_bid_fit`, `assess_bid_qualification`, `extract_bid_requirements`, `export_bid_shortlist`, `create_bid_calendar`, `create_bid_review_packet`, `draft_bid_questions`, `analyze_past_awards`) と、MCP Apps 対応クライアント向けの検索結果テーブルUI (`search_bids_app`, `ui://jp-bids/search-results.html`) を追加しています。Tools/Prompts/Resources/Resource Templates/Completion/Logging に加え、SBOM、再現可能ビルド、Registry検証、UI typecheck、契約テストを備えた参照実装です。

## English

JP Bids MCP v0.4.0 is out. It wraps Japan's KKJ government procurement search API as an MCP server, letting Claude, Cursor, and VS Code search public bid opportunities through natural language. v0.4.0 adds the AI Bid Radar tool suite (`rank_bids`, `explain_bid_fit`, `assess_bid_qualification`, `extract_bid_requirements`, `export_bid_shortlist`, `create_bid_calendar`, `create_bid_review_packet`, `draft_bid_questions`, `analyze_past_awards`) plus an MCP Apps-compatible search results table UI (`search_bids_app`, `ui://jp-bids/search-results.html`). It is designed as a reference-grade implementation with Tools, Prompts, Resources, Resource Templates, Completion, Logging, SBOM, reproducible builds, Registry validation, dedicated UI typechecking, and contract tests.
