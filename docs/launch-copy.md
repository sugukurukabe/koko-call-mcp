# Launch Copy

## 日本語

KokoCallMCP v0.2.0 を公開しました。中小企業庁 官公需情報ポータルサイトの検索APIをMCP化し、Claude/Cursor/VS Codeから日本全国の入札情報を自然言語で検索できます。Toolsだけでなく、Prompts、Resources、Resource Templates、Completion、Logging、SBOM、再現可能ビルド、Registry検証まで入れた参照実装を目指しています。

## English

KokoCallMCP v0.2.0 is out. It wraps Japan's KKJ government procurement search API as an MCP server, letting Claude, Cursor, and VS Code search public bid opportunities through natural language. It is designed as a reference-grade implementation with Tools, Prompts, Resources, Resource Templates, Completion, Logging, SBOM, reproducible builds, and Registry validation.

## Bahasa Indonesia

KokoCallMCP v0.2.0 telah dirilis. Server ini membungkus API pencarian pengadaan pemerintah Jepang KKJ sebagai MCP, sehingga Claude, Cursor, dan VS Code dapat mencari peluang tender publik dengan bahasa natural. Implementasi ini menargetkan kualitas referensi dengan Tools, Prompts, Resources, Resource Templates, Completion, Logging, SBOM, reproducible builds, dan validasi Registry.
