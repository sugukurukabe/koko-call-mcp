# 公共情報は公開されている。でも、使えるとは限らない。

入札情報は、そこにある。

中小企業庁の官公需情報ポータルサイトには、全国の公共調達情報が集まっている。検索APIもある。仕様書もある。つまり、形式上は公開されている。

けれど、現場で本当に使えるかどうかは別の話である。

公開情報は、ただ置かれているだけでは使えない。誰かが、検索できる形にし、出典を保ち、必要な文脈だけを取り出せるようにしなければならない。

JP Bids MCP は、そのために作った。

AIエージェントが、日本の公共調達情報をMCP経由で読めるようにする。目的は派手なデモではない。公開されている情報を、機械が誤解しにくい形に整えることだ。

顔を出して何かを語るより、こういうものをひとつずつ置いていく方が、自分には合っている。

仕様を読む。
実装する。
公開する。
運用できる形にする。

それだけで分かる人には分かる。

公共情報は、公開されている。

でも、使えるとは限らない。

その差を埋める仕事が、これから増えていくと思っている。
