# API Reference

## 日本語

KokoCallMCP は `http://www.kkj.go.jp/api/` にGETリクエストを送り、XMLを正規化して返します。`Query`、`Project_Name`、`Organization_Name`、`LG_Code` のいずれか1つ以上が必要です。

## English

KokoCallMCP sends GET requests to `http://www.kkj.go.jp/api/`, normalizes XML, and returns typed results. At least one of `Query`, `Project_Name`, `Organization_Name`, or `LG_Code` is required.

## Bahasa Indonesia

KokoCallMCP mengirim request GET ke `http://www.kkj.go.jp/api/`, menormalisasi XML, dan mengembalikan hasil bertipe. Minimal salah satu dari `Query`, `Project_Name`, `Organization_Name`, atau `LG_Code` wajib ada.
