# Prompt Injection Defense

## 日本語

入札件名、説明、添付資料名は外部データです。KokoCallMCP はそれらを命令として解釈せず、構造化データとして返します。

## English

Bid names, descriptions, and attachment names are external data. KokoCallMCP never treats them as instructions and returns them as structured data.

## Bahasa Indonesia

Nama tender, deskripsi, dan nama lampiran adalah data eksternal. KokoCallMCP tidak pernah memperlakukannya sebagai instruksi dan mengembalikannya sebagai data terstruktur.
