# JP Bids MCP デモ動画スクリプト（90秒）

> 壁の撮影用。Loom / QuickTime / OBS いずれでも可。
> 編集なし・字幕なしで撮って公開するだけでOK。

---

## セットアップ（撮影前）

1. Cursorを開く
2. jp-bids-mcp が接続済みであることを確認（`.cursor/mcp.json`）
3. 新しいチャットを開く
4. フォントサイズを大きめに（Cmd + = で拡大）
5. ウィンドウをフルスクリーンに

---

## スクリプト（読み上げ原稿 + 操作手順）

### 0:00〜0:08 ——導入

**[画面: Cursorのチャット画面]**

> 「JP Bids MCPを使って、官公需入札調査をAIに全部やってもらいます。」

---

### 0:08〜0:25 ——案件検索

**[入力するテキスト]**
```
鹿児島県のIT系役務入札を10件探して
```

**[待ち時間に読み上げ]**
> 「都道府県・区分・キーワードで絞り込めます。全国180万件から検索します。」

**[結果が出たら]**
> 「10件ヒットしました。次にAIでランキングします。」

---

### 0:25〜0:45 ——ランキング

**[入力するテキスト]**
```
この中からソフトウェア・保守・クラウドに近い案件を優先して5件に絞って
```

**[待ち時間に読み上げ]**
> 「キーワードマッチ・締切日・組織名から自動スコアリングします。」

**[結果が出たら]**
> 「スコア順に並びました。1位の案件の仕様書を確認します。」

---

### 0:45〜1:05 ——仕様書取得

**[入力するテキスト]**
```
1位の案件の仕様書PDFを取得して、参加資格と評価基準を整理して
```

**[待ち時間に読み上げ]**
> 「PDFを自動取得してGeminiで解析します。」

**[結果が出たら]**
> 「参加資格・必須要件・評価基準が構造化されました。」

---

### 1:05〜1:20 ——カレンダー登録

**[入力するテキスト]**
```
この案件の締切日をICSカレンダーファイルで出して
```

**[結果が出たら]**
> 「カレンダーファイルが生成されました。ダブルクリックでGoogleカレンダーやOutlookに登録できます。」

---

### 1:20〜1:30 ——まとめ

**[画面: ランディングページ https://mcp.bid-jp.com に切り替え]**

> 「検索・ランキング・PDF解析・カレンダー登録まで、全部会話で完結しました。」
> 「ベータ期間中は全機能無料です。mcp.bid-jp.com から始められます。」
