# Benchmarks

## 日本語

`npm run bench` はXML正規化と都道府県コード変換の基本性能を測ります。結果は標準エラーに出力されます。

## English

`npm run bench` measures baseline performance for XML normalization and prefecture code conversion. Results are printed to stderr.

## Bahasa Indonesia

`npm run bench` mengukur performa dasar normalisasi XML dan konversi kode prefektur. Hasil ditulis ke stderr.
