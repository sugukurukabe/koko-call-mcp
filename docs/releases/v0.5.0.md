# JP Bids MCP v0.5.0

## 日本語

v0.5.0 は、AI Bid Radar のPDF要件抽出を本実装に近づけるリリースです。

主な変更:

- `extract_bid_requirements` に `fetch_documents` と `target_uris` を追加。
- PDF/HTML fetcherを追加し、サイズ上限、MIME検証、timeout、redirect上限、SSRF対策、host allowlistを実装。
- MCP samplingによるHTML/text要件抽出を追加。
- `JP_BIDS_VERTEX_AI=1` でVertex AI Gemini direct modeを有効化し、PDF bytesをGemini `inlineData` として処理。
- source URI、final URI、SHA-256、size、MIME、mode、warnings、raw extraction textをstructuredContentへ追加。
- `docs/adr/0015-pdf-extraction.md` と `docs/pdf-extraction.md` を追加。

## English

v0.5.0 moves AI Bid Radar's PDF requirement extraction toward production implementation.

Highlights:

- Added `fetch_documents` and `target_uris` to `extract_bid_requirements`.
- Added a PDF/HTML fetcher with size limits, MIME validation, timeout, redirect limits, SSRF protection, and host allowlists.
- Added HTML/text requirement extraction through MCP sampling.
- Added Vertex AI Gemini direct mode via `JP_BIDS_VERTEX_AI=1`, passing PDF bytes as Gemini `inlineData`.
- Added source URI, final URI, SHA-256, size, MIME, mode, warnings, and raw extraction text to structuredContent.
- Added `docs/adr/0015-pdf-extraction.md` and `docs/pdf-extraction.md`.
