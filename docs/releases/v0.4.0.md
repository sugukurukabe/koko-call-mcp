# JP Bids MCP v0.4.0

## 日本語

v0.4.0 は、MCP Apps対応クライアント向けの検索結果テーブルUIを追加するリリースです。

主な変更:

- `search_bids_app` toolを追加。
- `rank_bids` toolを追加し、AI Bid Radar MVPとして入札候補の追うべき順を返す。
- `explain_bid_fit` toolを追加し、1件ごとの判断理由、リスク、確認チェックリストを返す。
- `assess_bid_qualification` toolを追加し、自社地域・カテゴリ・資格との適合性をMVP判定。
- `extract_bid_requirements` toolを追加し、PDF/Document AI連携前の安全な要件整理を返す。
- `export_bid_shortlist` toolを追加し、Google Sheets / Excel向けの社内検討CSVを返す。
- `create_bid_calendar` toolを追加し、提出期限・開札日などをICS形式で返す。
- `create_bid_review_packet` toolを追加し、1案件の社内検討メモMarkdownを返す。
- `draft_bid_questions` toolを追加し、発注者へ確認する質問書ドラフトを返す。
- `analyze_past_awards` toolを追加し、過去公告の発注機関頻度・カテゴリ偏り・月次トレンドを集計する競合レーダーを提供。
- `due_*` 入力の説明を「入札書提出期限」に修正し、`bid_due_alert` プロンプトで日付フィルタを明示。
- `ui://jp-bids/search-results.html` resourceを追加。
- `docs://agentic-cloud-roadmap` resourceを追加し、Google-managed MCP / Gemini Enterprise / Agentic Data Cloud準拠の拡張方針を公開。
- `docs://agentic-security-storage-readiness` resourceを追加し、Cloud Storage Smart Storage / Fraud Defense / Agent Platform security準拠のreadinessを公開。
- React/Vite single-file buildでMCP Apps用HTMLを生成。
- UI専用TypeScript typecheckとUI resourceの契約テストを追加。
- `downloadFile`、`sendMessage`、`openLink` によるhost連携を追加。
- Host capabilityでguardした `createSamplingMessage` AI briefを追加。
- `updateModelContext` と `requestDisplayMode` によるhost bridgeを追加。
- UI非対応クライアント向けにテキスト要約とstructuredContent fallbackを維持。

## English

v0.4.0 adds a search-results table UI for MCP Apps-compatible clients.

Highlights:

- Added the `search_bids_app` tool.
- Added the `rank_bids` tool to return bid candidates ordered by follow-up priority as the AI Bid Radar MVP.
- Added the `explain_bid_fit` tool for per-bid rationale, risks, and confirmation checklist.
- Added the `assess_bid_qualification` tool to assess fit against company regions, categories, and qualifications.
- Added the `extract_bid_requirements` tool for safe requirement triage before PDF / Document AI integration.
- Added the `export_bid_shortlist` tool for Google Sheets / Excel internal-review CSV.
- Added the `create_bid_calendar` tool to return submission and opening dates as ICS.
- Added the `create_bid_review_packet` tool to return a Markdown internal-review memo for one bid.
- Added the `draft_bid_questions` tool to return draft clarification questions for the buyer.
- Added the `analyze_past_awards` tool as a competitor radar over past notices.
- Clarified the `due_*` filters as the bid submission deadline and added explicit date hints in the `bid_due_alert` prompt.
- Added the `ui://jp-bids/search-results.html` resource.
- Added the `docs://agentic-cloud-roadmap` resource for Google-managed MCP / Gemini Enterprise / Agentic Data Cloud-aligned expansion.
- Added the `docs://agentic-security-storage-readiness` resource for Cloud Storage Smart Storage / Fraud Defense / Agent Platform security readiness.
- Generate MCP Apps HTML through a React/Vite single-file build.
- Added dedicated TypeScript typechecking for the UI and a UI resource contract test.
- Added host integration through `downloadFile`, `sendMessage`, and `openLink`.
- Added a `createSamplingMessage` AI brief guarded by host capabilities.
- Added host bridges through `updateModelContext` and `requestDisplayMode`.
- Kept text summary and structuredContent fallback for clients without UI support.
