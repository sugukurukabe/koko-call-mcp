# JP Bids MCP v0.6.x

## 日本語

v0.6.x は、MCP Apps UI を「検索結果表」から「AI Bid Workspace」に刷新し、PDF抽出連携を全ツールに展開するリリースシリーズです。

### v0.6.2

- `create_bid_calendar` に `fetch_documents` を追加し、PDF抽出済みの説明会日時をICSカレンダーに追加。
- PDF抽出対応ツールが 8/14 に到達。

### v0.6.1

- `explain_bid_fit` と `export_bid_shortlist` に `fetch_documents` を追加。
- CSV shortlist にPDF抽出の参加資格・提出期限等5列を追加。

### v0.6.0

- AI Bid Workspace UI: Priority Lane / Workbench / Action Dock / Evidence & Safety Panel の4ペイン構成。
- 4ペルソナに基づくUX設計: 営業責任者 / 入札事務担当 / 経営者 / 新人担当者。
- Action DockのボタンはsendMessageでAI Bid Radarツールを仕事の言葉で呼び出し。
- ダークモード対応CSS。

## English

v0.6.x redesigns the MCP Apps UI from a search-results table into an AI Bid Workspace and rolls out PDF extraction integration to all applicable tools.

### v0.6.2

- Added `fetch_documents` to `create_bid_calendar` for PDF-extracted briefing dates in ICS.
- 8 out of 14 tools now support PDF/HTML document extraction.

### v0.6.1

- Added `fetch_documents` to `explain_bid_fit` and `export_bid_shortlist`.
- Added 5 PDF extraction columns to the CSV shortlist.

### v0.6.0

- AI Bid Workspace UI with Priority Lane, Workbench, Action Dock, and Evidence & Safety Panel.
- UX designed for four personas: sales manager, bid administrator, executive, junior staff.
- Action Dock buttons invoke AI Bid Radar tools via sendMessage in task language.
- Dark mode CSS support.

## Bahasa Indonesia

v0.6.x mendesain ulang UI MCP Apps dari tabel hasil pencarian menjadi AI Bid Workspace dan menyebarkan integrasi ekstraksi PDF ke semua tool yang berlaku.

### v0.6.2

- Menambahkan `fetch_documents` ke `create_bid_calendar` untuk tanggal briefing hasil ekstraksi PDF di ICS.
- 8 dari 14 tool kini mendukung ekstraksi dokumen PDF/HTML.

### v0.6.1

- Menambahkan `fetch_documents` ke `explain_bid_fit` dan `export_bid_shortlist`.
- Menambahkan 5 kolom ekstraksi PDF ke CSV shortlist.

### v0.6.0

- UI AI Bid Workspace dengan Priority Lane, Workbench, Action Dock, dan Evidence & Safety Panel.
- UX dirancang untuk empat persona: manajer penjualan, admin tender, eksekutif, staf junior.
- Tombol Action Dock memanggil tool AI Bid Radar melalui sendMessage dalam bahasa tugas.
- Dukungan CSS dark mode.
