# KokoCallMCP 日本語README

## 日本語

KokoCallMCP は官公需情報ポータルサイト検索APIをMCP化するサーバーです。自治体ITベンダー、建設業、公共コンサルが自然言語で入札情報を確認できます。

## English

KokoCallMCP wraps the KKJ procurement search API as an MCP server. It helps local IT vendors, construction companies, and public-sector consultants inspect bid opportunities through natural language.

## Bahasa Indonesia

KokoCallMCP membungkus API pencarian pengadaan KKJ sebagai server MCP. Server ini membantu vendor IT lokal, perusahaan konstruksi, dan konsultan sektor publik memeriksa peluang tender dengan bahasa natural.
